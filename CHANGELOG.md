# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-25

### Added
- Initial release of Weatherly.
- Premium glassmorphic UI design.
- Real-time weather data from WeatherAPI.
- 3rd-day forecast view.
- Hourly forecast dashboard.
- Settings with temperature unit toggle (Celsius/Fahrenheit).
- Custom animated splash screen.
- Smooth side drawer navigation.

### Changed
- Improved search bar visibility and persistence.
- Updated support email.
- Finalized app branding as "Weatherly".
